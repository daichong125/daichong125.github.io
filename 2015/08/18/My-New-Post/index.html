下载 jdk 并完成安装

下载地址

jdk（java开发包或java开发工具，java Development Kit），针对不同的操作系统和位数（32位/64位）有不同的版本，对应自己电脑选择不同版本的jdk下载并安装。

以我的电脑为例：windows 64位操作系统，对应下载如下图：

安装过程记下jdk安装路径，例：D:\Program Files (x86)\Java\jdk1.8.0_31

配置java开发环境

在 控制面板->系统->高级系统设置->高级 中选择环境变量

在系统变量下

点新建，分别输入“JAVA_HOME”和jdk安装目录，本例填“D:\Program Files (x86)\Java\jdk1.8.0_31”

在PATH路径中加入jdk下的bin目录，本例为“D:\Program Files (x86)\Java\jdk1.8.0_31\bin”

新建变量名为“CLASS_PATH”,变量名jdk下的lib目录，为“.;D:\Program Files (x86)\Java\jdk1.8.0_31\lib”，其中，”.;”表示当前目录下

印证java环境是否搭载成功

打开命令行，输入 javac

输入 java

得到以上信息（用法参数提示信息）表示java开发环境搭载成功

jdk中的主要文件

在 D:\Program Files (x86)\Java\jdk1.8.0_31\bin 文件夹中包含有以下主要可执行文件。

编译器 java.exe ，将.java源程序编译为.class字节码文件（跨平台）；

解释器java.exe，可接受.class文件并启动java虚拟机执行之；

Applet查看器 appletviewer.exe，用来查看java的applet小程序；

压缩工具jar.exe，将相关类文件打包成一个文件；

文档制作工具javadoc.exe，能从源码注释中提取文档；

头文件（header）产生器 javah.exe；
